Eat Safe

This project utilizes API's such as Zomato's API, a geocode api from rapidapi. Allows users to search restaurants in their area or an area of their choosing. Then it will display restaurants giving the address, cuisine type, phone number, if they deliver, and their opening times. Future development is to add Covid-19 information for the area searched to give ease to those who are looking for restaurants that deliver.

Weather Dashboard

The Weather dashboard displays current weather status and forecast for the next 5 days. Uses API from openweathermap.org for the information of temperature, humidity, wind speed, and more. Users can search for location and it will be stored to the left.

Code Quiz

An assignment that was done for Coding Boot Camp. A short 5-question quiz that is timed. Every wrong answer will give a -10 second penalty off the timer. User is able to store scores after quiz is finished.
